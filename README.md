# Trading Signals Bot 🚀

A comprehensive trading signals bot with advanced technical indicators, pattern recognition, and automated signal generation for Crypto, Forex, and OTC markets.

## Features

### 📊 Technical Indicators (10)
1. **RSI** - Relative Strength Index
2. **MACD** - Moving Average Convergence Divergence
3. **Bollinger Bands** - Volatility indicator
4. **Stochastic Oscillator** - Momentum indicator
5. **CCI** - Commodity Channel Index
6. **Williams %R** - Oscillator
7. **Momentum** - Price momentum
8. **EMA** - Exponential Moving Average
9. **SMA** - Simple Moving Average
10. **ATR** - Average True Range

### 🎯 Trading Patterns (10)
1. Head and Shoulders
2. Double Top
3. Double Bottom
4. Ascending Triangle
5. Descending Triangle
6. Wedge Pattern
7. Flag Pattern
8. Pennant
9. Cup and Handle
10. Rectangle

### 📈 Market Support
- **Crypto**: BTC, ETH, XRP, ADA, SOL
- **Forex**: EUR/USD, GBP/USD, USD/JPY, AUD/USD, USD/CAD
- **OTC Stocks**: AAPL, GOOGL, MSFT, AMZN, TSLA
